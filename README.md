# Interactive Maths Quiz Game for Class 10

## 1. Description

This project is a command-line based interactive quiz game designed to help 10th-grade students practice key mathematics concepts in an engaging way. The application presents randomly generated questions from various topics relevant to the curriculum, tracks the user's score, and provides immediate feedback.

This project fulfills the first-semester coursework requirements by identifying a real-world need (making math practice enjoyable) and applying fundamental programming concepts to create a practical educational tool.

## 2. Features

-   **Multiple Topics:** The quiz includes questions from different areas of the Class 10 curriculum.
-   **Randomized Questions:** Questions are generated dynamically, ensuring a unique quiz experience each time.
-   **Score Tracking:** The application keeps track of the user's score and presents a final summary at the end.
-   **Immediate Feedback:** Users are told whether their answer was correct or incorrect right away.
-   **User-Friendly Interface:** A simple and intuitive menu system makes the game easy to navigate.

## 3. Topics Covered

The quiz currently generates questions from the following topics:
-   **Algebra:** Finding the roots of quadratic equations.
-   **Trigonometry:** Calculating the values of basic trigonometric ratios (sin, cos, tan) for standard angles.
-   **Arithmetic Progressions:** Finding the nth term of a sequence.

## 4. How to Run the Project

### Prerequisites
-   You must have Python 3 installed on your system.

### Steps
1.  **Clone the Repository:**
    Open your terminal or command prompt and clone this repository to your local machine:
    ```bash
    git clone <your-repository-url>
    ```

2.  **Navigate to the Directory:**
    Change into the project's directory:
    ```bash
    cd <repository-name>
    ```

3.  **Run the Script:**
    Execute the following command in your terminal to start the game:
    ```bash
    python quiz_game.py
    ```

4.  **Follow the on-screen instructions** to start the quiz and answer the questions.
